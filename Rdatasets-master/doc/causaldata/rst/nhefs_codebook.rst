NHEFS Codebook
--------------

Description
~~~~~~~~~~~

``nhefs_codebook`` is the codebook for ``nhefs`` and ``nhefs_complete``.

Usage
~~~~~

::

   nhefs_codebook

Format
~~~~~~

A data frame with 64 rows and 2 variables.

variable
   The variable being described

description
   The variable description

Source
~~~~~~

https://wwwn.cdc.gov/nchs/nhanes/nhefs/
